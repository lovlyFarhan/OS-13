
###grep

*grep*

    [solome@app ~]$ grep [-acinv -AB] [--color=auto] 'string' filename
    PARAMS
        -a  :  將二進制(BINARY)文件以文本(TEXT)文件的方式查找數據
        -c  :  計算查找字符串的次數
        -i  :  忽略大小寫
        -n  :  顯示行號
        -v  :  反向選擇
        --color=auto  :  在關鍵部分進行加亮顯示
        -A  :  AFTER[n] 後面的n行同樣也會顯示
        -B  :  BEFOEE[n] 前面等n行同樣也會顯示
